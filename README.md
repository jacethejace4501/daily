Daily Modpack
Daily is my personal everyday Minecraft setup — a lightweight, fast, and clean Fabric modpack built for survival, servers, building, and casual play. It keeps the game feeling completely vanilla while adding the performance boosts and quality‑of‑life tweaks I use every single day.

✨ What’s Inside

⚡ Performance Mods — smoother FPS, faster chunk loading, reduced lag

🎨 Visual Improvements — cleaner UI, better readability, subtle polish

🧭 Quality of Life — small tweaks that make the game feel better without changing gameplay

🧩 Lightweight & Compatible — works on any world or server

💻 Perfect for Daily Play — nothing intrusive, nothing bloated

🎯 Why Daily? — Because sometimes you just want Minecraft to run better, look cleaner, and feel smoother — without turning it into a different game. Daily is the setup I use every time I open Minecraft.

📦 Installation — Click Install in the Modrinth App and you’re ready to play.
